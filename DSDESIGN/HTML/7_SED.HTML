Запрос:

select K_TSID, K_BNID, CTS, DZS, DRS, K_ULID,BIC+'.'+CTS CODE ,CUL,BIC,NBN from K_TS join LME_PARAMS on K_ULID_INPUT=ID and NAME='KF_ID'

Запрос строки:

select K_TSID, K_BNID, CTS, DZS, DRS, K_ULID ,BIC+'.'+CTS CODE ,CUL,BIC,NBN from K_TS where K_TSID=^K_TSID^

Вставка:

exec LME_AddK_TS ^K_BNID^, ^CTS^, ^DZS^, ^DRS^, ^K_ULID^

Изменение:

exec LME_UpdK_TS ^K_BNID^, ^CTS^, ^DZS^, ^DRS^, ^K_ULID^, ^K_TSID^

Удаление:

delete from K_TS where K_TSID=^K_TSID^

Дополнительное условие:

order by CODE
