Бюджет 99

Запрос:

select PARAMSID, NAME, ID, VALUE from LME_PARAMS

Запрос строки:

select PARAMSID, NAME, ID, VALUE from LME_PARAMS where PARAMSID=^PARAMSID^

Вставка:

Insert Into LME_PARAMS ( NAME, ID, VALUE) Values ( ^NAME^, ^ID^, ^VALUE^)

Изменение:

Update LME_PARAMS Set NAME=^NAME^, ID=^ID^, VALUE=^VALUE^ Where PARAMSID=^PARAMSID^

Удаление:

delete from LME_PARAMS where PARAMSID=^PARAMSID^

Дополнительное условие:

Индекс по умолчанию:

order by NAME
