Общие данные

Соединение:

Бюджет 99

Запрос:

select 'Дата:' + CONVERT(VARCHAR(20),DATE,103)+' Счет:'+CTS CODE, BNVID, BNV.K_TDID, BNV.DOCID,BNV.K_TSID, CTS, BIC, NBN, DATE, DEBIT, CREDIT, INPUT, OUTPUT, MANUAL from BNV join K_TS on K_TS.K_TSID=BNV.K_TSID join LME_PARAMS on K_TS.CUL=VALUE and NAME='KFINN'

Запрос строки:

select 'Дата:' + CONVERT(VARCHAR(20),DATE,103)+' Счет:'+CTS CODE, BNVID, BNV.K_TDID, BNV.DOCID,BNV.K_TSID, CTS, BIC, NBN, DATE, DEBIT, CREDIT, INPUT, OUTPUT, MANUAL from BNV join K_TS on K_TS.K_TSID=BNV.K_TSID where BNVID=^BNVID^

Вставка:

LME_AddBnv ^DATE^, ^K_TSID^, ^DEBIT^,^CREDIT^, ^INPUT^,^OUTPUT^

Изменение:

Update BNV Set K_TSID=^K_TSID^, DATE=^DATE^, DEBIT=^DEBIT^, CREDIT=^CREDIT^, INPUT=^INPUT^, OUTPUT=^OUTPUT^, MANUAL=^MANUAL^ Where BNVID=^BNVID^

Удаление:

delete from BNV where BNVID=^BNVID^

Дополнительное условие:

Индекс по умолчанию:

order by DATE desc, CTS

Параметры

@K_TSID

Индексы

По счету:

order by CTS,DATE desc

По дате (убывание):

order by DATE desc,CTS

По дате (возрастание):

order by DATE,CTS
