Редактор: Классификация доходов

Общие данные

Соединение:

Бюджет 99

Запрос:

select K_DID, K_DID_PARENT, CD, CR, CP, CS, CRS, NS, CD+' '+NS CODE from K_D

Запрос строки:

select K_DID, K_DID_PARENT, CD, CR, CP, CS, CRS, NS, CD+' '+NS CODE from K_D where K_DID=^K_DID^

Вставка:

Insert Into K_D ( K_DID_PARENT, CD, CR, CP, CS, CRS, NS) Values ( ^K_DID_PARENT^, ^CR^+^CP^+^CS^+^CRS^, ^CR^, ^CP^, ^CS^, ^CRS^, ^NS^)

Изменение:

Update K_D Set K_DID_PARENT=^K_DID_PARENT^, CD=^CR^+^CP^+^CS^+^CRS^, CR=^CR^, CP=^CP^, CS=^CS^, CRS=^CRS^, NS=^NS^ Where K_DID=^K_DID^

Удаление:

delete from K_D where K_DID=^K_DID^

order by CD
