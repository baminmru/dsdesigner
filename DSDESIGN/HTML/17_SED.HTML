Запрос:

select BNV_LINESID, BNVID, N_STR, CDOC, CTS, BIC, SUMM from BNV_LINES

Запрос строки:

select BNV_LINESID, BNVID, N_STR, CDOC, CTS, BIC, SUMM from BNV_LINES where BNV_LINESID=^BNV_LINESID^

Вставка:

Insert Into BNV_LINES ( BNVID, N_STR, CDOC, CTS, BIC, SUMM) Values ( ^BNVID^, ^N_STR^, ^CDOC^, ^CTS^, ^BIC^, ^SUMM^)

Изменение:

Update BNV_LINES Set BNVID=^BNVID^, N_STR=^N_STR^, CDOC=^CDOC^, CTS=^CTS^, BIC=^BIC^, SUMM=^SUMM^ Where BNV_LINESID=^BNV_LINESID^

Удаление:

delete from BNV_LINES where BNV_LINESID=^BNV_LINESID^
