Общие данные

Соединение:

Бюджет 99

Запрос:

select N_STR, CDOC, CTS, BIC, BANK_SUMM, OST_SUMM, SUMM, DB_CBS, DB_NBS, DB_SUBCONTO, CR_CBS, CR_NBS, CR_SUBCONTO, BNV_LINESID, BNVID,EX_NUM from V_BNV_LINES_D2

Запрос строки:

select N_STR, CDOC, CTS, BIC, BANK_SUMM, OST_SUMM, SUMM, DB_CBS, DB_NBS, DB_SUBCONTO, CR_CBS, CR_NBS, CR_SUBCONTO, BNV_LINESID, BNVID,EX_NUM from V_BNV_LINES_D2 Where BNV_LINESID=^BNV_LINESID^ and EX_NUM=^EX_NUM^

Вставка:

Insert Into BNV_LINES ( BNVID, N_STR, CDOC, CTS, BIC, SUMM) Values ( ^BNVID^, ^N_STR^, ^CDOC^, ^CTS^, ^BIC^, ^BANK_SUMM^)

Изменение:

Update BNV_LINES Set BNVID=^BNVID^, N_STR=^N_STR^, CDOC=^CDOC^, CTS=^CTS^, BIC=^BIC^, SUMM=^BANK_SUMM^ Where BNV_LINESID=^BNV_LINESID^

Удаление:

delete from BNV_LINES where BNV_LINESID=^BNV_LINESID^

Дополнительное условие:

Индекс по умолчанию:

order by N_STR

Параметры

@BNVID

Индексы

По сумме выписки:

order by BANK_SUMM

По необработанной сумме:

order by OST_SUMM

По обработанной сумме:

order by SUMM

По проводке:

order by DB_CBS,DB_SUBCONTO,CR_CBS,CR_SUBCONTO

По кредиту проводки:

order by CR_CBS,CR_SUBCONTO

По дебету проводки:

order by DB_CBS,DB_SUBCONTO
