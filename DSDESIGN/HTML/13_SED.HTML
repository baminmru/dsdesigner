Общие данные

Соединение:

Бюджет 99

Запрос:

select N_STR, CDOC, CTS, BIC, BANK_SUMM, OST_SUMM, SUMM, DB_CBS, DB_SUBCONTO, CR_CBS, CR_SUBCONTO, EX_NUM,BNV_LINESID, BNVID from V_BNV_LINES_R2

Запрос строки:

select N_STR, CDOC, CTS, BIC, BANK_SUMM, OST_SUMM, SUMM, DB_CBS, DB_SUBCONTO, CR_CBS, CR_SUBCONTO, EX_NUM,BNV_LINESID, BNVID from V_BNV_LINES_R2 where BNV_LINESID=^BNV_LINESID^ and EX_NUM=^EX_NUM^

Вставка:

Insert Into BNV_LINES ( BNVID, N_STR, CDOC, CTS, BIC, SUMM) Values ( ^BNVID^, ^N_STR^, ^CDOC^, ^CTS^, ^BIC^, ^BANK_SUMM^)

Изменение:

Update BNV_LINES Set BNVID=^BNVID^, N_STR=^N_STR^, CDOC=^CDOC^, CTS=^CTS^, BIC=^BIC^, SUMM=^BANK_SUMM^ Where BNV_LINESID=^BNV_LINESID^

Удаление:

delete from BNV_LINES where BNV_LINESID=^ BNV_LINESID^

Дополнительное условие:

Индекс по умолчанию:

order by N_STR

Параметры

@BNVID

Индексы

По номеру строки:

order by N_STR

По проводке:

order by DB_CBS,DB_SUBCONTO,CR_CBS,CR_SUBCONTO

По дебету:

order by DB_CBS,DB_SUBCONTO

По кредиту:

order by CR_CBS,CR_SUBCONTO

По сумме:

order by BANK_SUMM desc

По номеру документа:

order by CDOC
