Общие данные

Соединение:

Бюджет 99

Запрос:

select K_ULID, CUL, NUL, K_UL.K_RFUID,CRFU,NRFU ,CUL+' '+NUL CODE from K_UL left join K_RFU on K_UL.K_RFUID=K_RFU.K_RFUID

Запрос строки:

select K_ULID, CUL, NUL, K_UL.K_RFUID,CRFU,NRFU ,CUL+' '+NUL CODE from K_UL left join K_RFU on K_UL.K_RFUID=K_RFU.K_RFUID where K_ULID=^K_ULID^

Вставка:

Insert Into K_UL ( CUL, NUL, K_RFUID ) Values ( ^CUL^, ^NUL^, ^K_RFUID^)

Изменение:

Update K_UL Set CUL=^CUL^, NUL=^NUL^, K_RFUID=^K_RFUID^ Where K_ULID=^K_ULID^

Удаление:

delete from K_UL where K_ULID=^K_ULID^

Индекс по умолчанию:

order by CRFU desc,CUL
